## Análisis inicial

La página nos da la bienvenida con el siguiente mensaje:

> Hay un camino oculto para encontrar la siguiente flag. A los robots no les gusta que se visite, pero quizá a ti sí...

En las páginas web, los archivos `robots.txt` son utilizados para indicar a los motores de búsqueda qué partes del sitio web no deben rastrear. Vamos a comprobar si existe este archivo en el sitio web.

```none title="/robots.txt"
User-agent: *
Disallow: /admin
Disallow: /logs
Allow: /
Disallow: /supersecretroute483284/index.html
```

El archivo `robots.txt` nos indica que hay una ruta llamada `/supersecretroute483284/index.html` que está prohibida para los robots. Vamos a intentar acceder a esa ruta.
