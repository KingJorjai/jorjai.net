## Descripción del desafío

Un mensaje misterioso ha aparecido en la pared de una cueva. Parece estar en un lenguaje antiguo, pero una pista dejada por un explorador te indica que no debes leer el mensaje, sino observar con detenimiento. El primer dígito del candado se esconde en una pista muy sutil.

## Análisis inicial

Con la primera escucha, queda muy claro que se trata de un mensaje en código Morse. Hay tonos cortos y largos, separados por pausas. Para decodificarlo, podemos utilizar una herramienta en línea como [Morse Code Adaptive Audio Decoder](https://morsecode.world/international/decoder/audio-decoder-adaptive.html).

![Un GIF mostrando cómo se utiliza la herramienta en línea para decodificar el audio en código Morse](./assets/morse-decode.gif)

Al subir el archivo de audio a la herramienta, obtenemos la siguiente decodificación:
```
I K E R L A N M 0 R S 3 C 0 D 3 G 0 3 S W 1 L D
```

El formato de la flag es `ikerlan{mensaje}`, todo en minúsculas. Por lo tanto, la flag completa es:

```
ikerlan{m0rs3c0d3g03sw1ld}
```
